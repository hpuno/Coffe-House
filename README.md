# Nhập Môn Lập Trình Web - CT188

## COFFEE HOUSE WEBSITE

### Giới thiệu

Đây là đồ án học phần CT188 được xây dựng nhằm mục tiêu áp dụng các kiến thức về phát triển ứng dụng web.
Hệ thống mô phỏng một website bán cà phê với các chức năng hiển thị sản phẩm, đánh giá người dùng và quản lý dữ liệu thông qua API giả lập.

### Mục tiêu

Xây dựng giao diện website bán hàng cơ bản
Làm việc với REST API (json-server)
Thực hành fetch dữ liệu bất đồng bộ
Triển khai ứng dụng lên môi trường online

### Công nghệ sử dụng

- HTML5 / CSS3 / JavaScript / Bootstrap 5/ JQUERY
- JSON Server (Fake REST API)
- Git & GitHub để quản lý mã nguồn
- Render / GitHub Pages để triển khai

### Chức năng chính

- Hiển thị danh sách sản phẩm cà phê (hạt & đồ uống)
- Xem thông tin chi tiết sản phẩm
- Hiển thị đánh giá khách hàng
- Kết nối API để lấy dữ liệu động
